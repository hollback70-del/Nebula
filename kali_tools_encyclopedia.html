📁 CORE AUDITING TOOLS (CATEGORIES)

1. Nmap (Network Mapper)
നെറ്റ്‌വർക്കിലെ ഹോസ്റ്റുകളും സർവീസുകളും പോർട്ടുകളും സ്കാൻ ചെയ്ത് കണ്ടെത്തുന്നു.
nmap -sV [target_ip]
2. Wireshark
ലൈവ് നെറ്റ്‌വർക്ക് പാക്കറ്റുകൾ ക്യാപ്‌ചർ ചെയ്യാനും അനലൈസ് ചെയ്യാനും സഹായിക്കുന്നു.
wireshark
3. Burp Suite
വെബ് ആപ്ലിക്കേഷനുകളിലെ സെക്യൂരിറ്റി പ്രശ്നങ്ങൾ ടെസ്റ്റ് ചെയ്യാനുള്ള പ്രോക്സി ടൂൾ.
burpsuite
4. John the Ripper
മറന്നുപോയ പാസ്‌വേഡുകൾ ക്രാക്ക് ചെയ്യാനും ഹാഷ് സുരക്ഷ പരിശോധിക്കാനുമുള്ള ടൂൾ.
john [password_hash_file]
5. Metasploit Framework
സെക്യൂരിറ്റി പിഴവുകൾ പരിശോധിക്കാനും സിസ്റ്റം സുരക്ഷ ഉറപ്പാക്കാനുമുള്ള കൂറ്റൻ ഡാറ്റാബേസ്.
msfconsole
6. Hydra
ലോഗിൻ പേജുകളുടെ സുരക്ഷാ കരുത്ത് പരിശോധിക്കാനുള്ള അതിവേഗ നെറ്റ്‌വർക്ക് ടൂൾ.
hydra -l admin -P pass.txt [ip] ssh

🗺️ NMAP COMMAND MANUAL (മലയാളം ഗൈഡ്)

നെറ്റ്‌വർക്ക് സെക്യൂരിറ്റി ഓഡിറ്റിംഗിനായി ഉപയോഗിക്കുന്ന Nmap കമാൻഡുകൾ താഴെ നൽകുന്നു:

Nmap കമാൻഡ് ഉപയോഗിക്കുന്നത് എന്തിനുവേണ്ടി? (Purpose)
nmap [target_ip] ഒരു കമ്പ്യൂട്ടറിലെ അല്ലെങ്കിൽ വെബ്‌സൈറ്റിലെ ഏറ്റവും പ്രധാനപ്പെട്ട 1000 പോർട്ടുകൾ സാധാരണ രീതിയിൽ സ്കാൻ ചെയ്യാൻ.
nmap -v [target_ip] Verbose Mode: സ്കാനിംഗ് നടക്കുമ്പോൾ ബാക്ക്ഗ്രൗണ്ടിൽ എന്താണ് സംഭവിക്കുന്നത് എന്ന് ലൈവായി സ്ക്രീനിൽ കാണാൻ.
nmap -p 80,443 [target_ip] നിർദ്ദിഷ്ട പോർട്ടുകൾ മാത്രം (ഉദാഹരണത്തിന് പോർട്ട് 80, 443 മാത്രം) പ്രത്യേകം സ്കാൻ ചെയ്യാൻ.
nmap -p- [target_ip] ഒരു സിസ്റ്റത്തിലുള്ള ആകെ 65535 പോർട്ടുകളും മുഴുവനായി സ്കാൻ ചെയ്ത് പരിശോധിക്കാൻ.
nmap -sV [target_ip] Version Detection: പോർട്ടുകളിൽ റൺ ചെയ്യുന്ന സോഫ്റ്റ്‌വെയറുകൾ ഏതാണെന്നും അവയുടെ വേർഷൻ ഏതാണെന്നും കണ്ടെത്താൻ.
nmap -O [target_ip] OS Detection: ടാർഗെറ്റ് സിസ്റ്റം ഉപയോഗിക്കുന്നത് വിൻഡോസ് ആണോ ലിനക്സ് ആണോ ആൻഡ്രോയിഡ് ആണോ എന്ന് തിരിച്ചറിയാൻ.
nmap -sS [target_ip] SYN Stealth Scan: ടാർഗെറ്റ് സിസ്റ്റത്തിലെ ലോഗുകളിൽ പെട്ടെന്ന് പെടാതെ വളരെ രഹസ്യമായി പോർട്ടുകൾ സ്കാൻ ചെയ്യാൻ.
nmap -A [target_ip] Aggressive Scan: ഒരൊറ്റ കമാൻഡ് വഴി OS, വേർഷൻ, ട്രേസ്‌റൂട്ട്, സ്ക്രിപ്റ്റ് സ്കാനിംഗ് എന്നിവ ഒന്നിച്ച് പ്രവർത്തിപ്പിക്കാൻ.
nmap -sn [target_ip_range] Ping Scan: നെറ്റ്‌വർക്കിൽ നിലവിൽ എത്ര കമ്പ്യൂട്ടറുകൾ ഓൺ ആയിട്ടുണ്ടെന്ന് (Live Hosts) മാത്രം കണ്ടെത്താൻ.
